# Odyssey DLC Blockers

Issues that prevent or significantly hinder playing Odyssey DLC with Hardcore SK.

## 1. Chem refinery requires reinforced concrete blocks

- **Severity:** Critical
- **Problem:** Chem refinery requires 25 reinforced concrete blocks to build. On orbital/gravship maps there is no way to obtain reinforced concrete — no research, no resources, no traders.
- **Proposed solutions:**
  1. Add 25 reinforced concrete blocks to Gravship scenario starting items
  2. Simplify the refinery recipe — remove reinforced concrete requirement
  3. Create a new simpler refinery variant for orbital use

## 2. Steam generator requires water

- **Severity:** Critical
- **Problem:** Steam generator requires water connection (DBH water system) which doesn't work on gravship/orbital maps. No water infrastructure available in space.

# Блокеры DLC Odyssey

Проблемы, которые мешают или значительно затрудняют игру в DLC Odyssey с Hardcore SK.

## 1. Переработчик химии требует железобетонные блоки

- **Критичность:** Критическая
- **Проблема:** Переработчик химии требует 25 железобетонных блоков на постройку. На орбитальных/гравикорабельных картах железобетон неоткуда взять — нет исследований, ресурсов, торговцев.
- **Предложения:**
  1. Дать 25 железобетонных блоков на старте в сценарии
  2. Упростить рецепт — убрать требование железобетона
  3. Сделать новый упрощённый перегонный аппарат для орбиты

## 2. Паровой генератор требует воду

- **Критичность:** Критическая
- **Проблема:** Паровой генератор требует подключение воды (система воды DBH), которая не работает на картах гравикорабля/орбиты. В космосе нет водной инфраструктуры.
